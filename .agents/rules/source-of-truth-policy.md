# Source of Truth Policy

## Truth Source Priority
1. Manufacturer diagram/manual
2. Supplier exploded view with diagram sections
3. Seeded provider route verified by operator
4. Supplier product/pricing page
5. Uploaded manual/PDF extraction
6. LLM extraction only as helper evidence, never sole truth source

## Required Rule
Expected total parts count must be tied to assembly sections/diagram sections, not inferred from AI text.
