# Skill: Human-Centric SEO Optimization

## Context
This skill enables agents to optimize product listings for local search visibility while maintaining an authentic, professional, and "human" brand voice. The output must avoid clinical or repetitive AI phrasing.

## Guidelines for "Human-Written" Feel
* **Vary Sentence Structure:** Avoid starting every bullet point with the same verb (e.g., "Features...", "Features..."). Use a mix of benefit-driven and feature-driven leads.
* **Local Nuance:** Integrate local landmarks, neighborhoods (e.g., Quinby, Effingham), or regional climate considerations (e.g., "Perfect for a humid SC garage").
* **Avoid Over-Optimization:** Do not repeat the model number or "Florence, SC" more than 3-4 times. Priority is readability first, crawlers second.
* **Conversational Logic:** If a unit has a scratch, describe it honestly. Humans trust "Small scuff on the side" more than "Minor cosmetic wear."

## Workflow Tasks
* **H1 Transformation:** Upgrade generic titles (e.g., "GE Fridge") to high-intent local titles (e.g., "GE 18.1 Cu. Ft. Top-Freezer Refrigerator in Bisque | Florence, SC").
* **Hyper-Local Linkage:** Research and append a "Serving the Community" section mentioning specific 15-mile radius towns.
* **Schema Validation:** Generate JSON-LD snippets for `Product` and `LocalBusiness` to enable Rich Snippets.
* **Alt-Text Authoring:** Write descriptive alt-text for images that includes the model, color, and condition.

## Execution Logic (The "Agentic" Loop)
1.  **Analyze:** Read the existing listing and identify missing "High-Value" keywords (Model #, Capacity, Color, Location).
2.  **Draft:** Rewrite the Technical Specifications and Key Features into the approved bulleted format.
3.  **Localize:** Inject 1-2 sentences of geographical context specific to the Pee Dee region.
4.  **Verify:** Run a "Natural Language Check"—if it sounds like a manual, rewrite it to sound like a shop owner.
